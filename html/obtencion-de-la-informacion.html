Documentación y Obtención de lnformación.

Como obtener y documentar de forma efectiva.

Obtencion de la informacion

Proceso básico para obtener información

  1. Definir el tema.
  2. Buscar fuentes.
  3. Revisar la confiabilidad.
  4. Seleccionar la información útil.
  5. Guardar los datos importantes.
  6. Registrar las fuentes consultadas.

Ruta de investigación

Paso 1: Definir el tema
Antes de buscar, se debe tener claro qué se quiere investigar.

Paso 2: Buscar fuentes
Se consultan libros, artículos, páginas confiables, videos educativos o documentos.

Paso 3: Revisar la confiabilidad
Se analiza si la fuente tiene autor, fecha, evidencia y relación con el tema.

Paso 4: Seleccionar información útil
Se eligen las ideas que realmente ayudan a explicar el tema.

Paso 5: Guardar datos importantes
Se registran conceptos, ejemplos, cifras o ideas principales.

Paso 6: Registrar fuentes
Se anotan los sitios, libros o documentos consultados.
